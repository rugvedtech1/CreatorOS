# General Creator Playbook (Fallback)

Use this file when the creator's niche doesn't match a specific playbook
in this folder (e.g. finance, fitness, fashion, gaming, comedy, spiritual,
automobile, etc.), or when their niche is a blend that doesn't fit neatly
into one category.

## Universal principles that apply regardless of niche

- **Specificity beats generality** — "fitness tips" is forgettable;
  "the exact mistake beginners make with squats" is save-worthy. This
  holds across every niche.
- **Real-world context still matters** — check `engines/daily-intelligence.md`
  for how to tie current weather/season/trends into this creator's
  specific niche, even without a dedicated playbook.
- **Format still matters more than niche** — Reels for reach/discovery,
  Carousels for save-worthy reference content, Stories for daily trust-
  building, apply universally.

## Building a working playbook for this creator on the fly

Since no dedicated file exists yet, gather the same signals a dedicated
playbook would define:
1. What does this creator's audience actually search for/struggle with?
2. What real-world events/seasons/trends naturally intersect with this
   niche?
3. What visual/format conventions already work well in this specific
   niche (ask the creator what they've seen perform well for others)?
4. What's the emotional core of this niche — is it aspirational,
   practical, entertaining, educational? This shapes hook/CTA choice
   more than the topic itself.

## Note for future development

If this creator's niche comes up repeatedly, consider adding a dedicated
file to `niches/` following the same structure as the other playbooks in
this folder (monitor list, content angles, format notes, guardrail note).
